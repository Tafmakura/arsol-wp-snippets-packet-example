Arsol WP Snippets Packet Example

This an example of what you can achieve with Arsol WP Snippets packets, this content is loaded from an external HTML template file added to the parket and is loaded via a shortcode, if you disable the "Example Package with Includes" file in the plugin settings, this content will disappear becaue the shortcode is run via the added packet

Cool Features

  • External HTML file inclusion
  • Separate styling through CSS
  • Modular content structure
  • Easy to debug, version control, maintain and update

More Benefits

  • Separation of concerns
  • Reusable components
  • Clean code organization
  • Easy to modify independently

Granular packet control demonstration

This section is red and blue when the nested CSS file "Example Nested CSS with Granular Control" is enabled, when the file is disabled the css classes will be disabled.

You can control the styling of this section by enabling or disabling the CSS file through the Arsol WP Snippets plugin.

When enabled, the word "red" will be red and the word "blue" will be blue.

When disabled, all text will be black.


How This Content Works:

  1. Content is generated through PHP functions in included files
  2. Styling is applied through a nested CSS file added via arsol_wp_snippets_css_addon_files filter
  3. CSS can be disabled granularly, allowing for complex combinations of styling
  4. Multiple included files can work together to create rich functionality
  5. HTML templates can be loaded from external files for better organization and maintainability

All this demonstrates how a packet can include its own files, provide functionality through shortcodes, manage styling through nested CSS files that can be enabled/disabled independently, and load HTML templates from external files.
